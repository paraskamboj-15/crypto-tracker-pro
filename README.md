# 🚀 Crypto Tracker Pro - Real-Time Analytics Dashboard

A modern, high-performance cryptocurrency tracking dashboard built with React and Tailwind CSS. It provides real-time market data, interactive financial charts, and advanced side-by-side coin comparisons.

## ✨ Key Features
- **Live Market Data:** Fetches real-time pricing, market cap, and volume using the CoinGecko REST API.
- **Advanced Data Visualization:** Interactive area charts and dual-axis line charts for coin comparisons using `recharts`.
- **Smart Comparison Engine:** Compare the performance of any two cryptocurrencies on a single unified graph with proportional data bars.
- **Premium UI/UX:** Pixel-perfect glassmorphism design, custom animations, skeleton loaders, and a fully responsive layout.
- **Optimized Performance:** Features client-side pagination, seamless search filtering, and parallel API data fetching using `Promise.all`.
- **Flawless Architecture:** Utilized React Portals for modal rendering to bypass CSS stacking context/scroll issues, and implemented robust event propagation handling.

## 🛠️ Tech Stack
- **Frontend Framework:** React.js (Vite)
- **Styling:** Tailwind CSS (v3)
- **Data Visualization:** Recharts
- **Icons & Assets:** Lucide React
- **API Client & Networking:** Axios
- **Data Source:** CoinGecko API (V3)
